Proyecto

Tecnología aplicada a la prevención de incendios

Este proyecto presenta un sistema inteligente capaz de detectar incendios mediante visión por cámara, identificar el tipo de fuego y activar automáticamente el extintor adecuado para su control.

Funcionalidades

¿Qué hace nuestro sistema?

Un sistema automatizado que detecta, analiza y actúa ante incendios.

Detección por Cámara

Analiza en tiempo real el entorno mediante visión artificial para identificar incendios.

Clasificación de Fuego

Determina el tipo de fuego (eléctrico o común) para tomar la mejor decisión.

Activación Automática

El sistema actúa sin intervención humana al detectar una situación de riesgo.

Extintor Adecuado

Selecciona entre dos tipos de extintores según el tipo de incendio detectado.

Monitoreo en Tiempo Real

Permite observar el estado del sistema y alertas en todo momento.

Prototipo Innovador

Proyecto enfocado en la automatización y seguridad mediante tecnología inteligente.
